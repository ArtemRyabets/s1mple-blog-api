# S1mpleBlog API

## Опис проекту
SimpleBlog — це бекенд-сервіс (REST API) для платформи публікації контенту. Проект розроблений для забезпечення базової функціональності блогу: створення статей, їх редагування та коментування.

Основна мета проекту — побудова надійної архітектури, яка здатна витримувати навантаження та легко розширюватися новими модулями.

### Ключові можливості
* Публікація: Створення та форматування статей (Title, Body, Tags).
* Взаємодія: Система коментування під кожним постом.

---

## Технологічний стек
Проект базується на сучасних інструментах розробки, обраних для забезпечення типізації, якості коду та швидкодії.

| Категорія | Технологія | Пояснення вибору |
| :--- | :--- | :--- |
| Runtime | Node.js | Висока продуктивність I/O операцій. |
| Language | TypeScript | Статична типізація для уникнення помилок. |
| Framework | Express.js | Мінімалістичний та гнучкий веб-фреймворк. |
| Database | PostgreSQL | Надійна реляційна база даних для зв'язків (Пости <-> Коментарі). |
| ORM | Prisma / TypeORM | Для зручної роботи з базою даних (буде обрано в Лаб 4). |
| Testing | Jest | Стандарт де-факто для тестування JS/TS додатків. |
| Linting | ESLint + Prettier | Забезпечення єдиного стилю коду. |

---


### Статті (Posts)
Сутність, що містить заголовок, контент, дату створення.

| Метод | URL | Опис |
| :--- | :--- | :--- |
| GET | /posts | Отримати список всіх статей (стрічка). |
| GET | /posts/:id | Отримати повний текст однієї статті. |
| POST | /posts | Створити нову статтю. |
| PATCH| /posts/:id | Оновити статтю (наприклад, виправити помилку в тексті). |
| DELETE| /posts/:id | Видалити статтю. |

### Коментарі (Comments)
Сутність, прив'язана до конкретної статті.

| Метод | URL | Опис |
| :--- | :--- | :--- |
| GET | /posts/:id/comments | Отримати всі коментарі до статті. |
| POST | /posts/:id/comments | Додати новий коментар. |
| DELETE| /comments/:id | Видалити коментар (модерація). |
